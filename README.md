# 🌤️ Weather App

A responsive weather application that lets users search for a city and view current weather conditions. Built as part of a Frontend Mentor challenge to practice working with APIs and dynamic UI updates.

## 🔗 Live Demo

👉 https://sykes6ix.github.io/Weather-App/

## 🚀 Features

* Search for weather by city name
* Displays temperature, weather condition, and location
* Responsive design for mobile and desktop
* Clean and simple user interface

> ⚠️ Note: Loading state and temperature unit toggle (°C/°F) are not implemented in this version.

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Weather API

---

## 📚 What I Learned

* Fetching and handling data from an external API
* Updating the DOM dynamically based on user input
* Structuring responsive layouts
* Handling errors like invalid city searches
